Google Analytics Measurement Protocol

A PHP Implementation of Google's Measurement Protocol's REST api

Basic Usage

As shown in the repository README, the basic usage would go something like this:

$ua = new \UniversalAnalytics\UA(array(
    'v' => 1,
    'tid' => 'UX-XXXX-XXX',
    'cid' => 555,
));

$event = new \UniversalAnalytics\Track\Event;
$event->category = 'Video';
$event->action = 'Play';
$event->label = 'Cat Video 42';
$event->value = '0';

$request = $ua->track($event);
$response = $request->send();

Alternatively, each tracked "entity" (such as a Page view, Event, or Transaction) has a "shortcut" method:

$ua = new \UniversalAnalytics\UA(array(
    'v' => 1,
    'tid' => 'UX-XXXX-XXX',
    'cid' => 555,
));

$request = $ua->event(array(
    'category' => 'Video',
    'action' => 'Play',
    'label' => 'Cat Video 42',
    'value' => '0',
))->track();

$response = $request->send();

Tracking Entities

The following entities are included in this library for tracking:

  • eCommerce Item
  • eCommerce Transaction
  • Event
  • Exception
  • Page
  • Social Interaction
  • User Timing

eCommerce Item

$ua = new \UniversalAnalytics\UA(...);

$item = new \UniversalAnalytics\Track\CommerceItemHit;
$item->id = '123456789';  // Transaction ID, REQUIRED
$item->name = 'Sofa';     // REQUIRED
$item->price = '2.99';
$item->quantity = '3';
$item->code = 'SKU_1234';
$item->variation = 'blue';
$item->currency_code = 'USD';

$request = $ua->track($item);
$response = $request->send();

Alternatively:

$ua = new \UniversalAnalytics\UA(...);

$request = $ua->item(array(
    'id' => '123456789', // Transaction ID, REQUIRED
    'name' => 'Sofa',    // REQUIRED
    'price' => '2.99',
    'quantity' => '3',
    'code' => 'SKU_1234',
    'variation' => 'blue',
    'currency_code' => 'USD',
))->track();

$response = $request->send();
